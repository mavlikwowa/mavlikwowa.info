There is my CV-project developed with [Next.js](https://nextjs.org/).

## Commands
Runs development server (it will be open on [http://localhost:3000](http://localhost:3000)):
```
yarn dev
```
Runs linter:
```
yarn lint
```
Runs spellcheck:
```
yarn yaspeller
```

## Data`s texts
Texts locate in "data" directory (RU/ENG).
